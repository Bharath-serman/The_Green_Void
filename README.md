# The Green Void

**The Green Void** is a small **3D narrative-driven horror game** currently in development using **Unity 2022.3.41 LTS**.  
The project focuses on atmosphere, psychological tension, and minimal storytelling rather than combat-heavy mechanics.

---

## Game Overview

- **Engine:** Unity 2022.3.41 LTS  
- **Genre:** Psychological Horror / Exploration  
- **Platform:** PC (initially)  
- **Status:** In Development  

The game follows a fragmented narrative where the player explores an abandoned environment and slowly uncovers the truth behind a mysterious phenomenon known as **The Green Void**.

---

## Core Themes

- Psychological horror  
- Isolation and uncertainty  
- Environmental storytelling  
- Reality distortion and hallucinations  

## Development Focus

- Minimal UI, maximum immersion  
- Use of Unity Timeline for cutscenes and dialogues  
- Lighting, sound design, and pacing as core horror tools  
- Optimized for low-to-mid range systems  

---

## Project Status

This project is:
- A **learning + portfolio project**
- Actively evolving
- Subject to changes in story, structure, and mechanics
